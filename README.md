## üöÄ Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/domedeve/dripzandkicks.git
   ```

2. Open `index.html` in your browser or serve locally:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   ```

3. Visit `http://localhost:8000`
